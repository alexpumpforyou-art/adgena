# AdGena RF Proxy

Этот каталог содержит конфигурацию для **прокси-сервера** в Яндекс Облаке,
который улучшает доступ к приложению из РФ.

**Не нужно деплоить эту папку на Railway!** Это отдельный самостоятельный сервис.

## Что делает

```
adgena.pro (DNS) → Yandex VPS (этот прокси) → Railway (приложение)
```

- Конечные пользователи в РФ обращаются на IP в РФ (быстро, без блокировок)
- Прокси перенаправляет все запросы на Railway-домен
- SSL через Let's Encrypt (Caddy делает всё автоматом)

См. `../PROXY_SETUP.md` для пошаговой инструкции.
